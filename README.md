# Netflix-Presentation
This project focuses on analyzing Netflix movies and shows data to uncover insights related to ratings, genres, runtime, age certification, and audience preferences. The objective was to clean, transform, and analyze raw datasets to derive meaningful business insights using SQL-based data analysis techniques.

**The analysis involved:**

• Identifying and removing duplicate and null values
• Joining Netflix content data with credit tables
• Extracting top-performing movies based on IMDb scores
• Analyzing runtime distribution
• Identifying top-rated and low-rated shows
• Genre-based filtering (Comedy, Action, Horror, Fantasy, etc.)
• Creating rating categories (Good, Bad, Excellent) using case studies
• Updating and handling missing age certification values
• Aggregating certification counts and performance metrics
